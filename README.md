# Análise Empírica de Complexidade de Algoritmos

Este projeto visa avaliar o tempo de execução de algoritmos de busca e ordenação, comparando-os com suas complexidades assintóticas teóricas ($O(n)$, $O(\log n)$, etc.).

## Algoritmos Implementados
- **Busca Sequencial**: $O(n)$
- **Busca Binária**: $O(\log n)$

## Como Compilar e Executar
É necessário ter o `g++` instalado. No terminal, execute:

```bash
g++ -Wall main.cpp SearchAlgorithms.cpp -o programa
./programa
